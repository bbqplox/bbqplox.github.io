Fizzle

Fizzle is an Android application built with Android Studio and Firebase. Users are able to generate drink recipes based on ingredients they already have and find other nearby users to share with using a swipe feature. The app features Google Login and uses the profile image from the user's Google account when displayed to other users. The main focus of the UI design involves minimalistic icons and consistent system status updating for easy navigation.

WhoseChore

WhoseChore is a web application developed using Ruby on Rails framework and PostgreSQL. It helps housemates manage chore responsibilities through the application's chore rotation feature. Coupled with email reminders, getting tasks done has never been easier.

WhoseChore was developed using agile software development principals. My role during the project was being the software architect. During the coding sprints, my main contributions for the project was implementing the main functionality of the web application and its interaction with the database.

Check out the demo. Check out the website.
